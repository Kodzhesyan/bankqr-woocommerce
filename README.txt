Додає спосіб оплати через IBAN із генерацією QR-коду для українських банків.

== Description ==

Bank-QR Payment Gateway додає можливість оплати через IBAN з автоматичною генерацією QR-коду для швидкої оплати через мобільні додатки українських банків.

Основні можливості:
* Генерація QR-коду для оплати
* Автоматичне перенаправлення на сторінку оплати
* Підтримка всіх українських банків
* Зручний інтерфейс налаштувань
* Повна інтеграція з WooCommerce

== Installation ==

1. Завантажте плагін в директорію `/wp-content/plugins/`
2. Активуйте плагін через меню 'Плагіни' в WordPress
3. Налаштуйте платіжний шлюз в WooCommerce → Налаштування → Платежі

== Frequently Asked Questions ==

= Як отримати API ключ? =

Зареєструйтесь на сайті bank-qr.com.ua та отримайте API ключ у особистому кабінеті.
